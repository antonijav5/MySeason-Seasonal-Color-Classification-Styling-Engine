# MySeason 🎨

> Discover your personal color season through intelligent image analysis.

MySeason is a desktop/mobile application that analyzes the dominant color
from a photo and classifies it into one of the **12 color seasons** used
in personal color analysis (PCA). Built with Python, Kivy, OpenCV, and
scikit-learn.

## Features

- 📷 **Smart color extraction** — filters out skin tones, backgrounds,
  and neutral colors to isolate the most relevant chromatic color
- 🎯 **12-season classification** — scoring-based engine covering all
  12 seasons across Spring, Summer, Autumn, and Winter families
- 📊 **Score visualization** — top 3 season results displayed as
  animated progress bars
- 🖼️ **Season icons** — each season has its own visual identity
- 🎨 **Muted season palette** — UI adapts its color scheme to the
  detected season

---

## The 12 Seasons

| Family | Seasons |
|--------|---------|
| 🌸 **Spring** | Light Spring · True Spring · Bright Spring |
| 🌊 **Summer** | Light Summer · True Summer · Soft Summer |
| 🍁 **Autumn** | Soft Autumn · True Autumn · Dark Autumn |
| ❄️ **Winter** | Bright Winter · True Winter · Dark Winter |
